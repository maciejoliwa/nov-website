Getting started

A quick FAQ about the language:

Why would you use this language instead of Python with BS4?

No idea honestly, nov's interpreter is written in Python and uses BS4 so it just abstracts certain operation and makes some quality of life improvements.

Hello World

A simple hello world program consits of show function

show "Hello World!"

That's it, now run it: python3 nov.py file.nov

If statements

Ifs can only be done inline (for now)

age = 19
naughty_things = "no idea"

show naughty_things if age >= 18

Functions

Take note that for now function arguments are not supported!

function say_hello do
    show "Hello!"
end

say_hello

Scraping

URL = "https://<some-website>.com"
scrapped_website = scrap URL

p_elements = get-all-html scrapped_website "p"
img_elements = get-all-html scrapped_website "img"

show p_elements
show img_elements

Saving to a file

You can save anything into a file, scrapped data, strings, numbers, you name it.

"hello" |> "hello.txt"

URL = "https://<some-website>.com"
scrapped_website = scrap URL

scrapped_website |> "index.html"

Loops

Loops are like 10% implemented, but you can use repeat like so:

// this will show "hello!" 100 times
repeat 100 show "hello!"
